# 🎬 Movie Finder – A Simple Python Tool

This is a command‑line program that lets you search for movies and get details (plot, rating, year, genre) using a free public API – **no sign‑up, no API key, no personal information required**.

The script works on **Windows, macOS, and Linux**.

---

## 📦 What you need before starting

### 1. Python 3.14 (or newer)

The script is written in Python. You need to have Python installed on your computer.

**Why Python 3.14?**  
Python 3.14 is the latest version (as of 2026). Older versions (like 3.6 or 3.10) will also work, but installing the newest version ensures you have all the latest features and security updates. The script runs on any Python 3.6+.

**How to check if you have Python:**  
- Open **Command Prompt** (Windows) or **Terminal** (Mac/Linux).  
- Type `python --version` and press Enter.  
- If you see `Python 3.14.4` (or similar), you're ready.  
- If you see an error, download Python from [python.org](https://python.org/downloads/) – make sure to check **“Add Python to PATH”** during installation.

### 2. The `requests` library

This is a small Python library that lets our script talk to the internet. You install it **once** using the terminal.

## 🚀 How to set up and run the Movie Finder

### Step 1 – Download the script

- Click on `movie_finder.py` in this repository.  
- Click the **“Download raw file”** button (or copy the code manually).  
- Save the file somewhere easy to find, e.g., your **Desktop** or a folder named `MovieFinder` inside `Documents`.

### Step 2 – Open a terminal (command prompt)

**On Windows:**  
Press `Win + R`, type `cmd`, press Enter.

**On macOS:**  
Press `Cmd + Space`, type `Terminal`, press Enter.

**On Linux:**  
Open your distribution's terminal (usually `Ctrl + Alt + T`).
